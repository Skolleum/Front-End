# Mobile-App

## Stack

- Flutter + 2.5
- Truffle
- Ganache

## How to run

Truffle:

- Open ganache and add truffle.js file to ganache test project.
- in the project directory run:
  - truffle compile
  - truffle migrate

Flutter: - flutter pub get - flutter run
